# PBM Cost & Utilization Dashboard

**All data used in this project is fully synthetic and created solely for demonstration purposes.**

This project is a two‑page Power BI dashboard designed to demonstrate analytics engineering skills using a synthetic Pharmacy Benefit Management (PBM) dataset. It highlights cost drivers, utilization patterns, and therapeutic class performance through clean, executive‑ready visuals.

---

## 📊 Project Overview

The goal of this dashboard is to provide a clear, data‑driven view of pharmacy plan performance. It includes high‑level KPIs, cost trends, and a deeper analysis of therapeutic class drivers. The project showcases:

- Data modeling  
- DAX calculations  
- Visual design  
- Dashboard layout and UX polish  
- Documentation and version control  

This is a portfolio artifact intended to reflect real‑world PBM analytics work.

---

## 🗂️ Dashboard Pages

### **1. Cost & Utilization Overview**
A high‑level summary of plan performance, including:

- **Total Cost**  
- **PMPM (Per Member Per Month)**  
- **Claims Count**  
- **Average Cost per Claim**  
- **Cost Trend Over Time**

This page provides a quick, executive‑friendly snapshot of overall plan health.

---

### **2. Therapeutic Class Analysis**
A deeper dive into cost drivers by therapeutic class, featuring:

- **Top 5 Therapeutic Classes by Total Cost**  
- **Percent of Total Cost (Donut Chart)**  
- **Cost Trend for Top Classes**  
- **Filters for Therapeutic Class and Year**

This page helps identify which classes contribute most to spend and how they trend over time.

---

## 🧠 Key Metrics & Definitions

| Metric | Description |
|--------|-------------|
| **Total Cost** | Total pharmacy spend for the selected period |
| **PMPM** | Cost normalized per member per month |
| **Claims Count** | Number of processed pharmacy claims |
| **Avg Cost per Claim** | Total cost divided by claim volume |
| **% of Total Cost** | Share of total spend by therapeutic class |

---

## 🏗️ Data Model

The dataset includes standard PBM fields such as:

- Claim Date  
- Therapeutic Class  
- Drug Name  
- Quantity  
- Days Supply  
- Allowed Amount  
- Member ID  

The model uses a simple star‑schema structure with a Date dimension and a Claims fact table.
